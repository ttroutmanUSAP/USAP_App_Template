1. create new app dir
2. run command 'npx create-next-app'
3. copy server.js to new root (preset to port 5000, can be changed...)
4. copy components, lib, pages, and public to new root and overwrite...
5. update package.json to reflect the new app name
6. run 'npm install' to install the app dependencies
7. run 'npm run dev' and navigate to https://localhost:5000 (or whatever port you changed it to)

Auth is in place. All functional components using hooks
